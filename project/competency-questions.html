SPARQL queries:

In this section of the project, eleven different queries will be performed on our knowledge graph. Each question is expressed in the SPARQL language according to the RDF model and the STOP ontology. Here, you can find the jupyternotebook of the SPARQL queries

  • #1 Which are the topics of the songs?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_n WHERE { ?song rdf:type stop:Song; stop:is_about ?topic. ?topic stop:have_name ?topic_n }

    Result: Death, Eternal Sleep, Nature, Dancing, Dream, Music, Nature, Suffering, Devastation, Love, Nature, Salvation, Suffering, Damnation, Hate, Justice, Knowledge, Religion, Salvation, Sickness, Suffering, Damnation, Knowledge, Salvation, Sickness, Suffering, Darkness vs light, Dream, Sight, Sky, Suffering, Damnation, Dream, Religion, Salvation, Death, Dream, Memory



  • #2 Which are the topics of the poems?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_name WHERE { ?poem rdf:type stop:Poem; stop:is_about ?topic. ?topic stop:have_name ?topic_n. }

    Result: Darknessvslight, Night, Sight, Sky, Church, Justice, Knowledge, Religion, Suffering, Dancing, Music, Nature, Sky, Drinking, Nature, Secret, Suffering, Darknessvslight, Dream, Memory, Music, Hate, Justice, Religion, Death, EternalSleep, Nature, Family, Marriage, Damnation, Hate, Religion, Salvation



  • #3 Which is the most frequent topic in general?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_n (COUNT(?unit) as ?number_of_units) WHERE { ?topic stop:appears_in ?unit. ?topic stop:have_name ?topic_n } GROUP BY ?topic ORDER BY desc(?number_of_units) LIMIT 1

    Result: Suffering



  • #4 Which is the most frequent topic in songs?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_n (COUNT(?unit) as ?number_of_songs) WHERE { ?topic stop:appears_in ?unit. ?unit rdf:type stop:Song. ?topic stop:have_name ?topic_n. } GROUP BY ?topic ORDER BY desc(?number_of_songs) LIMIT 1

    Result: Dream



  • #5 Which is the most frequent topic in poems?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_n (COUNT(?unit) as ?number_of_poems) WHERE { ?topic stop:appears_in ?unit. ?unit rdf:type stop:Poem. ?topic stop:have_name ?topic_n. } GROUP BY ?topic ORDER BY desc(?number_of_poems) LIMIT 1

    Result: Religion



  • #6 Which are the topics treated in the song "Un chimico"?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_n WHERE { ?unit rdf:type stop:Song; stop:have_name "Un chimico"^^rdfs:Literal; stop:is_about ?topic. ?topic stop:have_name ?topic_n. }

    Result: Devastation, Love, Nature, Salvation, Suffering



  • #7 Which are the topics treated in the poem "Trainor the druggist"?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_n WHERE { ?unit rdf:type stop:Poem; stop:have_name "Trainor, the druggist"^^rdfs:Literal; stop:is_about ?topic. ?topic stop:have_name ?topic_n. }

    Result: Family, Marriage



  • #8 Which songs/poems are associated with the topic "Sky"?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?unit_name WHERE { VALUES ?type { stop:Poem stop:Song } ?unit rdf:type ?type; stop:have_name ?unit_name; stop:is_about stop:Sky. }

    Result: Dippold, The Optician; Fiddler Jones; Un ottico



  • #9 In which songs/poems can we find the topic "Love"?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?unit_name WHERE { VALUES ?type { stop:Poem stop:Song } ?unit rdf:type ?type; stop:have_name ?unit_name; stop:is_about stop:Love. }

    Result: Un chimico



  • #10 Which characters of The hill are not mentioned in La collina?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT DISTINCT ?p_name WHERE { ?song stop:has_character ?s_character. ?s_character stop:have_name ?s_name. ?song stop:related_to ?poem. ?song stop:have_name "La collina"^^rdfs:Literal. ?poem stop:has_character ?p_character. ?p_character stop:have_name ?p_name. FILTER (NOT EXISTS {?song stop:has_character ?p_character}) }

    Result: Aunt Emily, Edith, Major Walker, Sevigne Houghton, Towny Kincaid, Uncle Isaac



  • #11 Which is the name that the character of the song "Un giudice" has in the original poem by Masters?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?poem_name WHERE { ?unit stop:have_name "Un giudice"^^rdfs:Literal; rdf:type stop:Song; stop:related_to ?poem. ?poem stop:have_name ?poem_name. }

    Result: Judge Selah Lively
