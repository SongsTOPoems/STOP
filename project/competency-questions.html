SPARQL queries:

In this section of the project, ten different queries will be performed on our knowledge graph. Each question is expressed in the SPARQL language according to the RDF model and the STOP ontology. Here, you can find the jupyternotebook of the SPARQL queries

  • #1 Which are the topics of the songs?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_n WHERE { ?song rdf:type stop:Song; stop:is_about ?topic. ?topic stop:have_name ?topic_n }

    Result: #1, #2



  • #2 Which are the topics of the poems?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_name WHERE { ?poem rdf:type stop:Poem; stop:is_about ?topic. ?topic stop:have_name ?topic_n }

    Result: #1, #2



  • #3 Which is the most frequent topic in general?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_n (COUNT(?unit) as ?number_of_units) WHERE { ?topic stop:appears_in ?unit. ?topic stop:have_name ?topic_n } GROUP BY ?topic ORDER BY desc(?number_of_songs) LIMIT 1

    Result: #1, #2



  • #4 Which is the most frequent topic in songs?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_n (COUNT(?unit) as ?number_of_songs) WHERE { ?topic stop:appears_in ?unit. ?unit rdf:type stop:Song. ?topic stop:have_name ?topic_n. } GROUP BY ?topic ORDER BY desc(?number_of_songs) LIMIT 1

    Result: #1, #2



  • #5 Which is the most frequent topic in poems?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_n (COUNT(?unit) as ?number_of_poems) WHERE { ?topic stop:appears_in ?unit. ?unit rdf:type stop:Poem. ?topic stop:have_name ?topic_n. }

    Result: #1, #2



  • #6 Which are the topics treated in the song "Un chimico"?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_n WHERE { ?unit rdf:type stop:Song; stop:have_name "Un chimico"^^rdfs:Literal; stop:is_about ?topic. ?topic stop:have_name ?topic_n. }

    Result: #1, #2



  • #7 Which are the topics treated in the poem "Trainor the druggist"?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?topic_n WHERE { ?unit rdf:type stop:Poem; stop:have_name "Trainor, the druggist"^^rdfs:Literal; stop:is_about ?topic. ?topic stop:have_name ?topic_n. }

    Result: #1, #2



  • #8 Which songs/poems are associated with the topic "Sky"?

    The query PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#> PREFIX stop: <http://www.semanticweb.org/martinapensalfini/ontologies/2022/9/stop#> SELECT ?unit WHERE { VALUES ?type { stop:Poem stop:Song } ?unit rdf:type ?type; stop:is_about stop:Sky. }

    Result: #1, #2
